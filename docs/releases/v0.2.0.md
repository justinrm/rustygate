# RustyGate v0.2.0

`v0.2.0` opens the Responses-first OpenAI compatibility track.

## Highlights

- Added `POST /v1/responses` with non-streaming and SSE streaming responses.
- Added OpenAI-shaped endpoints for embeddings, moderations, images, audio, files, batches, fine-tuning jobs, and realtime session creation.
- Aligned chat completion IDs with OpenAI-style `chatcmpl-*` public identifiers.
- Updated `GET /v1/models` to serialize OpenAI-shaped model objects by default.
- Added a compatibility matrix and optional official OpenAI Python SDK smoke script.

## Validation

Run the standard project checks:

```sh
cargo fmt
cargo clippy --all-targets --all-features
cargo test
```

Optional SDK smoke check:

```sh
python scripts/openai_sdk_smoke.py
```
