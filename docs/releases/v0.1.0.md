# RustyGate v0.1.0

Release date: 2026-05-03

## Baseline capabilities

- `GET /health` and `GET /ready`
- `POST /v1/chat/completions` with validation and provider routing
- Real provider support for `openai_compatible` and `anthropic`
- Fallback across retryable provider errors
- `GET /stats` and `GET /stats/providers` aggregate metrics
- Structured JSON errors with request IDs
- Optional SQLite persistence for request logs and provider attempts

## Validation performed

- `cargo fmt -- --check` (required formatting adjustments were applied with `cargo fmt`)
- `cargo clippy --all-targets --all-features -- -D warnings`
- `cargo test`
- `./scripts/smoke.sh`

All checks passed.

## Known limitations in v0.1.0

- No streaming chat completions
- No gateway authentication on chat and stats endpoints
- No gateway rate limiting or abuse throttling
- No provider circuit breaker state management

## Smoke script usage

```sh
./scripts/smoke.sh
```

Use a custom endpoint with:

```sh
BASE_URL=http://127.0.0.1:8080 ./scripts/smoke.sh
```
